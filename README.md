# ビジコン（ビジネスコンテスト）

## 概要

本リポジトリは、ビジネスコンテスト（ビジコン）への提出資料をまとめたものです。

中小企業向けクラウドベース業務効率化SaaSプラットフォームの事業計画を提案します。

## 提案内容

**事業名**: 中小企業向け統合型業務効率化SaaS

**ミッション**: すべての中小企業にシンプルで強力なデジタルツールを届ける

### 主な特徴

- **オールインワン統合** - プロジェクト管理・CRM・請求/経理を1つのプラットフォームで提供
- **AI搭載** - 業務データからのインサイト自動生成
- **価格競争力** - 個別ツール3つ分の半額以下
- **日本市場最適化** - 日本の商習慣（請求書、見積書、稟議）に完全対応

### 事業目標（3年計画）

| 指標 | 1年目 | 2年目 | 3年目 |
|------|-------|-------|-------|
| ARR（年間経常収益） | 1.2億円 | 4.8億円 | 12億円 |
| 有料顧客数 | 200社 | 700社 | 1,500社 |
