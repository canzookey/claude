# 株式会社ビジコン

## 会社概要

中小企業向けクラウドベース業務効率化SaaSプラットフォームを提供。

**ミッション**: すべての中小企業にシンプルで強力なデジタルツールを届ける

**ビジョン**: 2030年までに国内中小企業向けSaaS市場においてトップ3のポジションを確立する
