# 株式会社ビジコン - 資料作成リポジトリ

## 概要
このリポジトリは株式会社ビジコンの営業・提案資料を効率的に作成するためのものです。
テンプレートとスクリプトを使い、Claudeと連携して高品質な資料を素早く作成できます。

## ディレクトリ構成
```
templates/          # パワポテンプレート
  【ご紹介資料】株式会社ビジコン.pptx  # メインテンプレート
scripts/            # 資料生成スクリプト
  generate_slides.py  # スライド生成ツール
output/             # 生成した資料の出力先
```

## テンプレート構造（【ご紹介資料】株式会社ビジコン.pptx）

全18スライド構成：

| スライド | 種類 | 内容 |
|---------|------|------|
| 1 | 表紙 | 会社名・資料タイトル |
| 2 | 目次 | CONTENTS（01 会社概要、02 ご支援実績） |
| 3 | セクション区切り | 「会社概要」 |
| 4 | 会社紹介 | 会社情報（左）+ 3事業紹介（右） |
| 5 | 支援実績一覧 | ロゴ + 実績テーブル |
| 6 | 代表経歴 | プロフィール + 経歴テーブル |
| 7 | セクション区切り | 「IT・AI伴走支援」 |
| 8-10 | 事例スライド | 課題→支援内容→成果の構成 |
| 11 | セクション区切り | 「IT・AIシステム開発支援」 |
| 12-17 | 事例スライド | 課題→支援内容→成果の構成 |
| 18 | お問い合わせ | 連絡先 |

### 事例スライドの共通構造
各事例スライド（8-10, 12-17）は以下の統一フォーマット：
- **タイトル**: 事例名（例：「事例：DX推進室伴走支援（老舗カステラ製造会社様）」）
- **サブタイトル**: 成果の要約（1-2行）
- **業種タグ**: 右上に業種ラベル（製造業、建設業、不動産 等）
- **課題**: 顧客が抱えていた問題点（箇条書き）
- **支援内容**: ビジコンが行った支援の詳細
- **成果**: 定量的な成果（数値を含む）

## 資料作成の使い方

### 1. 事例スライドの追加
```bash
python3 scripts/generate_slides.py add-case \
  --title "事例：〇〇（△△会社様）" \
  --subtitle "成果の要約文" \
  --industry "製造業" \
  --challenge "課題の説明" \
  --support "支援内容の説明" \
  --result "成果の説明"
```

### 2. 提案資料の新規作成
```bash
python3 scripts/generate_slides.py new-proposal \
  --client "クライアント名" \
  --theme "提案テーマ"
```

### 3. テンプレートのカスタマイズ
テンプレートのデザイン要素（色、フォント、レイアウト）は元のpptxファイルから自動継承されます。
